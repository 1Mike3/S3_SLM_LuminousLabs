* Language : English
* Suffix "Test" for testfiles
* Comments are written using the JavaDoc standard
* Unit tests

    * GameFactoryImplTest
    * GameIOImpl
  * methods names are short descriptions of the test like [method under test}\_[input]\_[outcome] ...
    * createGame_validInput_returnsGame
